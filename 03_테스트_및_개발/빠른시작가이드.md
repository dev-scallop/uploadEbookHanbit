# 🚀 전자책 검토자 자동 등록 시스템 - 빠른 시작 가이드

## 📋 요약
이 시스템은 Google Play Console에서 전자책 검토자를 엑셀 파일 기반으로 자동 등록하는 프로그램입니다.

---

## ⚡ 빠른 시작 (3단계)

### 1️⃣ 프로그램 실행
Windows에서 다음 중 하나를 실행하세요:
- **실행.bat** (더블클릭)
- **실행.ps1** (PowerShell에서 실행)
- 또는 직접: `python "[개발중] 전자책자동등록.py"`

### 2️⃣ 엑셀 파일 준비
다음 형식의 엑셀 파일(.xlsx 또는 .csv)을 준비하세요:

| 이름 | 도서명 | 지메일 |
|------|--------|--------|
| 김철수 | Python 프로그래밍 기초 | test1@gmail.com |
| 이영희 | 데이터 분석 입문서 | test2@gmail.com |

### 3️⃣ 자동 등록 실행
1. GUI에서 엑셀 파일 선택
2. 데이터 미리보기 확인
3. "등록 시작" 버튼 클릭
4. Chrome에서 Google 계정 로그인
5. 자동 등록 진행 확인

---

## 🛠️ 시스템 요구사항

### 필수 요구사항
- ✅ **Windows 10/11**
- ✅ **Python 3.7+** ([다운로드](https://www.python.org/downloads/))
- ✅ **Google Chrome** 브라우저
- ✅ **Google Play Console** 접근 권한

### 자동 설치되는 패키지
실행 스크립트가 자동으로 다음 패키지를 설치합니다:
- `pandas` (엑셀 파일 처리)
- `openpyxl` (Excel 지원)
- `selenium` (브라우저 자동화)
- `webdriver-manager` (Chrome 드라이버)
- `requests`, `beautifulsoup4` (웹 처리)

## 🔧 문제 해결

### ❌ Python이 설치되지 않은 경우
```
오류: Python이 설치되어 있지 않습니다.
```
**해결방법**: [Python 공식 사이트](https://www.python.org/downloads/)에서 Python 3.7+ 설치

### ❌ 패키지 설치 실패
```
패키지 설치 중 오류가 발생했습니다.
```
**해결방법**: 
1. 관리자 권한으로 실행
2. 인터넷 연결 확인
3. 방화벽 설정 확인

### ❌ Chrome 드라이버 오류
```
드라이버 설정 실패
```
**해결방법**:
1. Chrome 브라우저 최신 버전 업데이트
2. `chrome_user_data` 폴더 삭제 후 재실행

### ❌ 로그인 실패
```
로그인 시간 초과
```
**해결방법**:
1. Chrome에서 수동으로 Google 계정 로그인
2. 2단계 인증 완료
3. "로그인 상태 유지" 체크
