# 🚀 빠른 시작 가이드

## 1️⃣ 설치 (1분)

```bash
# 패키지 설치
pip install -r requirements.txt
```

## 2️⃣ 실행 (10초)

```bash
# GUI 실행
python gui.py
```

## 3️⃣ 설정 (2분)

### ✅ 필수 입력 사항
1. **OpenAI API Key** 🔑
   - https://platform.openai.com/api-keys
   - `sk-proj-...` 형식

2. **HCTI User ID & API Key** 🖼️
   - https://htmlcsstoimage.com/
   - 회원가입 → Dashboard → API 키 확인

3. **Apps Script URL** 📊 (선택)
   - Google Sheets에 쓰기 권한 필요 시
   - 설정 방법은 README.md 참조

4. **저장 폴더** 💾
   - "선택" 버튼으로 폴더 지정
   - 예: `C:\Users\사용자\Desktop\결과물`

### 💾 저장
- 모든 입력 후 **"💾 설정 저장"** 버튼 클릭
- `.env` 파일에 자동 저장
- 다음 실행 시 자동 로드

## 4️⃣ 실행 방법

### 🎯 한 번 실행 (권장)
```
[▶ 한 번 실행] 버튼 클릭
→ 현재 시점의 새로운 행 처리
→ 완료 후 자동 중지
```

**사용 시기**:
- 테스트할 때
- 특정 행만 처리하고 싶을 때
- 수동으로 제어하고 싶을 때

### 🔄 연속 실행
```
[🔄 연속 실행] 버튼 클릭
→ 60초마다 자동으로 새 행 체크
→ [⏸ 중지] 버튼으로 중지
```

**사용 시기**:
- 장시간 모니터링
- 자동화 필요 시
- 여러 행이 계속 추가될 때

## ❓ 문제 해결

### "응답 없음"이 나와요
✅ **해결됨!** v2.1부터 백그라운드 실행으로 즉시 반응합니다.

### 스크롤이 나와요
✅ **해결됨!** v2.1부터 한 화면에 모두 표시됩니다.

### 이미지에 빈 항목이 나와요
✅ **해결됨!** v2.1부터 자동으로 기본값이 입력됩니다.

### OpenAI API 오류
```
❌ OpenAI API 호출 실패
```
**해결**:
1. API 키 확인 (sk-proj-로 시작하는지)
2. API 잔액 확인 (https://platform.openai.com/account/billing)
3. 네트워크 연결 확인

### HCTI 오류
```
❌ HTML to image service failed
```
**해결**:
1. HCTI User ID/API Key 확인
2. HCTI 잔액 확인 (무료: 50 이미지/월)
3. 네트워크 연결 확인

### Apps Script 401 오류
```
❌ 401 Client Error: Unauthorized
```
**해결**:
1. 웹앱 배포 확인
2. 액세스 권한: **"모든 사용자"** 설정 확인
3. URL 정확히 복사했는지 확인
