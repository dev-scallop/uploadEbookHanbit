## 📝 1단계: Python 설치 확인

### 무엇을 하나요?
컴퓨터에 Python이 설치되어 있는지 확인합니다.

### 명령어
PowerShell을 열고 다음을 입력하세요:

```powershell
python --version
```

### 예상 결과
```
Python 3.8.10
```
또는
```
Python 3.10.5
```

### 문제가 생기면?
- "python을 찾을 수 없습니다" 오류가 나면:
  1. https://www.python.org/downloads/ 에서 Python 설치
  2. 설치 시 **"Add Python to PATH"** 체크박스를 반드시 선택
  3. 설치 후 PowerShell을 **다시 시작**
  4. 다시 `python --version` 실행

## 🔧 3단계: 패키지 설치 (시스템 Python 사용)

### 무엇을 하나요?
시스템 Python에 프로젝트에 필요한 라이브러리들을 설치합니다.

### ⚠️ 가상환경을 사용 중이라면
먼저 가상환경을 비활성화하세요:
```powershell
deactivate
```

프롬프트에서 `(venv)` 표시가 사라지면 성공입니다.

### 명령어 (순서대로 실행)
```powershell
python -m pip install --upgrade pip
```
```powershell
python -m pip install --no-cache-dir -r requirements.txt
```

**주의**: Playwright 설치 후 추가 단계가 필요합니다:
```powershell
playwright install chromium
```

### 소요 시간
약 3~5분

### 예상 결과
- 많은 텍스트가 빠르게 지나갑니다
- 마지막에 "Successfully installed ..." 메시지

### 진행 상황
설치가 진행되는 동안 다음과 같은 메시지가 보입니다:
```
Collecting jinja2==3.1.2
  Downloading jinja2-3.1.2...
Installing collected packages: ...
Successfully installed ...
```

### 문제가 생기면?
- 권한 오류 발생 시:
  ```powershell
  python -m pip install --user --no-cache-dir -r requirements.txt
  ```
  이후:
  ```powershell
  playwright install chromium
  ```

## 🎨 4단계: 첫 페이지 생성 (테스트)

### 무엇을 하나요?
샘플 도서 데이터로 상세페이지를 자동 생성합니다.

### 명령어
```powershell
python main.py --source test --count 1
```

### 소요 시간
약 5~10초 (AI 기능 없이)

### 예상 결과

2024-01-15 10:30:45 [INFO] 총 1개 도서 처리 시작
...
2024-01-15 10:30:50 [INFO] 페이지 생성 완료: ...
2024-01-15 10:30:52 [INFO] JPG 변환 완료: ...
============================================================
생성 완료! 총 1개 페이지

## 👀 5단계: 결과 확인

### HTML 파일 보기

1. 생성된 파일 목록 확인:
```powershell
dir .\output\html\
```

2. 브라우저로 열기:
```powershell
Start-Process .\output\html\<파일명>.html
```

예시:
```powershell
Start-Process .\output\html\9788901234567_테스트도서1파이썬프로그래밍.html
```

### JPG 이미지 보기

1. 이미지 목록 확인:
```powershell
dir .\output\images\
```

2. 이미지 열기:
```powershell
Start-Process .\output\images\<파일명>.jpg
```

### 탐색기로 보기
```powershell
explorer .\output
```
이 명령으로 output 폴더가 탐색기에서 열립니다.

## ❓ 자주 발생하는 문제

### 1. "ModuleNotFoundError: No module named 'xxx'"
**원인**: 패키지가 설치되지 않았음

**해결**:
```powershell
# 패키지 재설치 (python과 pip 경로 불일치 방지)
python -m pip install --no-cache-dir -r requirements.txt
```

### 2. "python을 찾을 수 없습니다"
**해결**: Python 설치 후 PowerShell 재시작

### 3. "permission denied" 또는 "액세스 거부"
**해결**: PowerShell을 관리자 권한으로 실행

### 4. Playwright 실행 오류
**해결**:
```powershell
playwright install chromium
```

### 5. OpenAI API 오류
**원인**: API 키가 잘못되었거나 `.env` 파일이 없음

**해결**:
1. `.env` 파일이 프로젝트 루트에 있는지 확인
2. 파일 내용에 `OPENAI_API_KEY=sk-...` 형태로 키가 있는지 확인
3. API 키가 유효한지 OpenAI 웹사이트에서 확인

### 6. "ModuleNotFoundError: No module named 'openai'"
**원인**: openai 패키지가 현재 Python에 설치되지 않음 (pip/python 경로 불일치 가능)

**해결**:
```powershell
python -m pip install --no-cache-dir "openai>=1.3.0,<2.0.0"

# 설치 확인
python - << 'PY'
import sys
import openai
print('openai version:', openai.__version__)
print('python exe    :', sys.executable)
PY
```
