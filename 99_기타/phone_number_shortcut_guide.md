# 전화번호 인식 후 문자 발송 단축어 사용 가이드

## 단축어 기능 설명
이 단축어는 다음과 같은 기능을 수행합니다:
1. 사용자가 단축어를 실행하면 전화번호 입력 방식을 선택합니다(카메라 스캔 또는 직접 입력)
2. 카메라 스캔 선택 시:
   - 카메라가 열리고 사용자가 전화번호가 포함된 화면을 촬영합니다
   - 이미지에서 OCR(광학 문자 인식)을 통해 전화번호를 추출합니다
3. 직접 입력 선택 시:
   - 사용자가 문자를 보낼 전화번호를 직접 입력합니다
4. 사용자가 담당자 목록에서 한 명을 선택합니다
5. 선택한 담당자 정보와 함께 형식화된 메시지를 작성합니다
6. 인식 또는 입력된 전화번호로 문자를 발송합니다

## 사용 방법
1. 단축어를 실행합니다
2. 전화번호 입력 방식 선택 메뉴에서 "카메라로 전화번호 스캔" 또는 "직접 전화번호 입력"을 선택합니다
3. 카메라 스캔 선택 시:
   - 카메라가 열리면 전화번호가 있는 부분을 선명하게 촬영합니다
   - 전화번호 인식이 완료되면 다음 단계로 진행합니다
4. 직접 입력 선택 시:
   - 문자를 보낼 전화번호를 입력합니다 (기본 형식: 010-)
5. 담당자 선택 메뉴가 표시됩니다
6. 아래 담당자 중 한 명을 선택합니다:
   - 김성삼 책임 010-3233-4321
   - 김태진 책임 010-2582-1877
   - 임현기 책임 010-3827-9725
   - 이성훈 책임 010-8536-3253
   - 김주성 책임 010-6481-0664
   - 이정훈 책임 010-3897-9167
   - 길진철 부장 010-2213-7482
   - 이윤형 부장 010-7415-4321
7. 선택 후 자동으로 문자 작성 화면이 표시됩니다
8. 발송을 눌러 문자를 전송합니다

## 문자 메시지 형식
문자는 다음과 같은 형식으로 발송됩니다:
```
[한빛아카데미]
담당 영업자
(선택한 담당자 정보)
감사합니다
```

## 단축어 설정 방법
1. JSON 파일을 .shortcut 파일로 변환합니다
2. 아이폰으로 해당 파일을 전송하여 단축어 앱에서 열거나 import합니다

## 주의사항
- 촬영 시 전화번호가 잘 보이도록 화면에 가깝게 촬영하세요
- 조명이 좋은 환경에서 사용하면 인식률이 높아집니다
- 전화번호 인식에 실패한 경우 "전화번호를 찾을 수 없습니다" 알림이 표시됩니다
- iOS 15 이상에서 최적의 성능을 발휘합니다
